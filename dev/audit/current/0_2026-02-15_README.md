# Audit -- Current

Active audit reports for the current phase of work. When a phase is complete, the admin archives reports to `legacy/`.

## Types of Reports in This Folder

- Architecture audits with component-by-component verdicts
- Risk registers with impact/likelihood matrices
- Readiness audits for phase gates
- Incident post-mortems after failures

## Naming Convention

`{number}_{YYYY-MM-DD}_{DESCRIPTION}.md`

- Number is sequential and chronological (0, 1, 2, ...)
- Date is the creation date in ISO format
- Description is uppercase, underscore-separated
