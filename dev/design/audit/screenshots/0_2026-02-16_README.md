# Design -- Audit -- Screenshots

Capture directory for visual screenshots taken during design audits.

## Purpose

Screenshots are captured by Puppeteer MCP or similar browser automation tools during visual design reviews. They serve as evidence in design audit reports and track UI state over time.

## Usage

Reference screenshots in audit reports using relative paths:
`![Description](../../design/audit/screenshots/1_2026-02-16_HOMEPAGE_BEFORE.png)`

## Naming Convention

`{number}_{YYYY-MM-DD}_{DESCRIPTION}.{png,jpg,webp}`
