# Design -- Resources

Static visual assets: icons, fonts, logos, and illustrations.

## Purpose

Centralized storage for visual assets referenced by reports, components, and documentation. Keeps assets organized and prevents duplication across the project.

## Suggested Subdirectories

Create as needed:
- `icons/` -- SVG or PNG icon files
- `fonts/` -- Custom font files (WOFF2 preferred)
- `logos/` -- Brand logos in various formats

## Naming Convention

`{number}_{YYYY-MM-DD}_{DESCRIPTION}.{ext}`
