# Reports — Draft

Work-in-progress reports. Claude writes draft reports here.

## Structure

- `html/` — HTML formatted drafts
- `written/` — Markdown/text formatted drafts

## Naming Convention

`{number}_{YYYY-MM-DD}_{DESCRIPTION}.{ext}`

- Number is sequential and chronological (0, 1, 2, ...)
- Date is the creation date in ISO format
- Description is uppercase, underscore-separated
- Use `.html` in `html/` and `.md` in `written/`
