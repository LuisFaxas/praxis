# Reports

Structured reporting pipeline with draft and published stages.

## Structure

- `draft/` — Work-in-progress reports
  - `html/` — HTML formatted drafts
  - `written/` — Markdown/text drafts
- `published/` — Finalized reports (admin-only)
  - `html/` — Published HTML reports
  - `written/` — Published written reports

## Rules

- Claude creates reports in `draft/` only.
- **Only the admin moves reports to `published/`.**

## Naming Convention

`{number}_{YYYY-MM-DD}_{DESCRIPTION}.{ext}`

- Number is sequential and chronological
- Date is the creation date in ISO format
- Description is uppercase, underscore-separated
